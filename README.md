## Descrição Geral
A seguinte aplicação desenvolvida é um pequeno projeto para consolidar os conceitos acerca do **paradigma orientado a objeto**. É destinado a toda comunidade, visando estudar, utilizar cotidianamente e implementar melhorias no projeto. O mesmo visa abrir chamados pelos atores funcionários e comunidade, e em relação ao administrador, listar, atualizar, remover e fechar os devidos chamados realizados, principalmente na área de TI. Uma versão meramente compacta e muito simplificada do GLPI.

## Funcionalidades
- Abrir chamado (ator: funcionário)
- Selecionar chamados
- Alterar status chamado
- Editar campus chamado
- Excluir chamado

## Tecnologias Utilizadas
- **Python 3.11**
- **POO**
- **Pandas**
